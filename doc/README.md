java植物大战僵尸 面向对象oop与设计模式design-pattern(大概用了十几种 )练习
{: id="20210414105822-ce3ikcg"}

一开始只是想练习一下面向对象与设计模式 ，后来干脆写完了 ，javafx写的很麻烦，如果一开始就想写成这样的话绝对去学unity

保证学过oop的都能简单的看懂代码，文档在doc文件夹中（内有双向链接）
